## 1. 诊断：骨架很好，血肉不足

设计的工程纪律与理论框架（MDA / 动词优先 / 三重资源分离 / 状态组件）都在水准之上。问题不在"对不对"，而在"爽不爽"：

| # | 核心问题 | 后果 |
|---|----------|------|
| D1 | **开局 30 秒即体验峰值**：一局约 28 次有效操作（≈每 4.3s 一次），部署完就是"等冷却→看自动战斗" | 乐趣曲线一路衰减，玩家沦为观众 |
| D2 | **战场是死的，防御不会思考**：墙塔陷阱全是静态二进制，解法可照抄 | 是"解题"不是"博弈"，第三局开始腻 |
| D3 | **Combo 按图索骥**：文档为每将预设 1–2 条"正确连招" | 玩家在执行设计师的答案，而非发明打法 |
| D4 | **手感反馈几乎为零**：无连击/震屏/慢动作/音效阶梯 | 水浒"千军万马、快意恩仇"的燃落空，像"特战队渗透" |

**六维好玩画像（当前评分 /10）**：

```
F1 决策密度   ████████░░  8   ✅ 强（实时部署+技能+撤兵）
F5 进步可感   ██████░░░░  6   中（三星+声望解锁，线性）
F6 叙事意义   █████░░░░░  5   中（拨济闭环已设计但单薄）
F2 创造空间   ████░░░░░░  4   ⚠️ 弱（8选8 + 预设 combo，无 build）
F3 意外时刻   ██░░░░░░░░  2   ❌ 缺口（静态防御+无环境）
F4 手感反馈   █░░░░░░░░░  1   ❌ 缺口（几乎未设计）
```

**结论**：F2/F3/F4 合计 7/30，正好对应两大死因——"第一局不够爽"（F4）与"第三局开始腻"（F2/F3）。主攻这三个维度。

## 2. 改进方向（四组 + 长尾）

### A. 让手忙起来 —— 填等待真空（补 F1）

| 方案 | 内容 | 成本 |
|------|------|------|
| **战旗指令** | 集火 / 后撤 / 冲锋 / 散开 4 个群体指令，短冷却 8–15s、耗粮草 5–10，短暂覆盖单位 AI 而不取代它。操作量 ~28 → 50+ 次/局；逻辑层仅加一个"指令覆盖层"（约 200–300 行） | 低 |
| **战场随机事件** | 每 30–45s 一个：援军抵达 / 粮仓着火（90s 内抢救 or 放弃）/ 暴雨突降（火系失效）/ 梁山号角（粮草临时 +20）。同一关永不重复 | 低-中 |

> 附带收益：战旗指令中的"后撤"可顺势补上评审 P0-4 指出的**撤兵动词未落地**欠账。

### B. 让每局不一样 —— 治背板（补 F2/F3）

| 方案 | 内容 | 成本 |
|------|------|------|
| **技能变体（战场专长）** | 每局每名上场武将从 3–4 个专长池随机带 1 个（林冲：突进留火墙 / 破甲 -60% / 距离 +2）。8 将 × 4 专长 → 组合空间万级，玩家会拼出设计师没预见的 combo。仅需 `hero_skills.csv` 加 `variant_pool` 字段 | 低 |
| **梁山号令** | 战前三选一全局加成：粮草先行 / 火攻计 / 夜行衣 / 招安令……制造 build 路线（火攻号令 + 林冲火墙专长 + 公孙胜燃烧加成 = 火系 build） | 低 |
| **环境交互** | 天气（雨克火油、雾压射程、夜缩守军视野）+ 可互动物体（火药桶引爆、滚石推落、吊桥改路径）。题材契合"火烧草料场""夜袭曾头市" | 中 |
| **防御微适应 / 指挥官 AI** | 最小版：一条规则"哪路被破墙最多，庄丁优先增援该路"→ 玩家立刻获得"声东击西"博弈。完整版：L2 起每关主将每 30s 一个指挥动作（史文恭调兵、闻焕章重置陷阱、高廉屏障、蔡京增援） | 中-高 |

### C. 让爽感兑现 —— 兑现水浒幻想（补 F4）

| 方案 | 内容 | 成本 |
|------|------|------|
| **Juiciness 四层反馈** | ① 单位级：击杀粒子+0.1s 震屏、受击闪白后仰、暴击金字放大；② Combo 级：连击计数器（3/5/10 连）、高连击屏边红光；③ 事件级：破墙 0.3s 慢动作、塔倒塌压伤周围（又制造涌现）、核心摧毁全屏白闪+2s 慢动作；④ 星级级：三星逐颗点亮、音效升半音。**纯表现层，零平衡风险** | 中 |
| **英雄时刻特写** | 李逵旋风三杀 / 花荣秒杀塔 / 林冲终结核心最后 10% → 1.5s 慢镜特写。确认"你刚才那一下很帅" | 中 |
| **连锁破坏演出** | 墙破木屑伤敌、塔折断倒塌、粮仓着火蔓延相邻建筑、核心逐块崩解 | 中 |

### D. 修复乐趣曲线 —— 结构性大改（收益最大、成本最高）

**三幕战斗结构 + Boss 守将**：

```
幕一·破门（0-40s）     幕二·巷战（40-90s）      幕三·斩首（90-150s）
外墙+箭塔              内墙+庄丁群+资源建筑      核心+Boss 守将+限时护盾
选突破方向             清场/劫掠，新部署点解锁    all-in，限时破盾压力
```

- 每幕切换由"墙被破"触发，自然形成起承转合，根治 D1 的单峰衰减。
- Boss 守将替代"纯建筑核心"：L2 史文恭（高机动专杀远程 → 需嘲讽吸火）、L3 梁中书（召唤杂兵 → 需 AOE）、L4 高廉（驱散+黑风 → 需时迁打断）、L5 蔡京+禁军统领（双 Boss 综合考验）。同时解决"摧毁核心太抽象"的体验空洞。

### E. 长尾钩子（通关后还回来）

- **残局复盘**：失败时给破局点分析（"差 1 格 / 差 5 秒 / 差 1 个技能"），把失败变学习。
- **破釜沉舟**：核心被毁前最后 10s 可激活全员狂暴（DPS+50%、HP 持续衰减）——"差点就赢了"的传奇时刻。
- **招降系统**：击败守军英雄可招降为梁山武将（耗声望 + 拨济 +10%）——把"打败的敌人"变成"可玩内容"。
- **好汉事迹**：花式通关副目标（"时迁不触发任何陷阱通关"），喂收集型玩家。
- **每日/周常挑战 + 异步守寨**：限定条件（"本周禁用远程"）逼出新 combo；玩家自建山寨布局供他人劫寨，UGC 内容循环。

## 3. 性价比矩阵

```
 好玩收益 ↑
  高 │ 技能变体●  战旗指令●        Juiciness●
     │ 梁山号令●   环境系统●    三幕+Boss●
  中 │ 残局复盘●   长尾系统●    指挥官AI●
  低 │             叙事分支●
     └──────────────────────────────────→ 工程成本
            低              中              高
```

| 梯队 | 方向 | 定位 |
|------|------|------|
| 🥇 首批（低成本高收益） | 战旗指令、技能变体、梁山号令、Juiciness | MVP 增强，与 Phase 2/3 同步 |
| 🥈 第二批（中成本高收益） | 环境系统、三幕结构+Boss、指挥官 AI | 涌现提升，治"第三局腻" |
| 🥉 第三批 | 残局复盘、破釜沉舟、招降/事迹/挑战、叙事分支 | 长尾留存 |

## 4. 落地建议：如果只做三件事

1. **战旗指令**（成本最低、ROI 最高）——把"部署完看戏"变成"全程指挥"，操作量翻倍；顺带补齐撤兵机制欠账。
2. **技能变体 + Juiciness**（决定第一印象）——每局随机专长解决创造空间；击杀震屏/连击/慢动作/星级音效是纯表现层改动，零平衡风险，直接决定试玩留存。
3. **三幕结构 + Boss 守将**（收益最大但要排期）——根治乐趣曲线衰减，同时兑现水浒的"燃"（史文恭、高廉等原著反派登场）。

之后依次推进：环境系统 + 指挥官 AI（治背板）→ 复盘/招降/挑战（长尾）。
