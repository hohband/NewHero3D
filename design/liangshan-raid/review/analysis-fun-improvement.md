# 「劫寨」战斗系统 —— 好玩性提升分析

> 日期：2026-07-20
> 基于：combat-mode-design.md + 4 GDD + 涌现性审查

---

## 0. 诊断：当前设计的三个"不好玩"缺口

### 缺口一：决策密度过低

一局 120s 的核心循环中，玩家能做多少次有意义操作？

| 操作类型 | 估算次数 | 占比 |
|---------|---------|------|
| 部署武将 | ~8 次 | 主要在开局 30s |
| 释放主动技能 | ~20 次（8 将 × ~2.5 次/局） | 分散但间隔 5-20s |
| **合计** | **~28 次/120s** | **≈ 每 4.3 秒 1 次操作** |

对比：
- Hades / Dead Cells：每秒 2-5 次操作
- 皇室战争：每 5-10s 一张牌，但持续微操调整
- 自走棋 (TFT)：回合间频繁换位、合装备

**结论：玩家更多是"观众"而非"参与者"。** 单位自动寻路/索敌/攻击，技能冷却 10-22s，大部分时间玩家在等待。

### 缺口二：乐趣曲线衰减

```
乐趣 ↑
    │  ● 开局部署（高决策密度）
    │    ╲
    │      ╲  第一波技能 combo（爽）
    │        ╲
    │          ╲________________ 等待冷却、看自动战斗（无聊）
    │                             ╲
    │                                ╲  摧毁核心（结束）
    └─────────────────────────────────────→ 时间
    0s         30s         60s         90s        120s
```

开局 30s 是体验峰值，之后就是"等冷却→放技能→看动画→再等"的重复循环。没有攀升、没有转折、没有高潮。

### 缺口三：水浒幻想落空

水浒传的核心情绪是什么？**一百零八好汉、千军万马、替天行道、快意恩仇**。

当前设计给了什么？12 个单位上限、2.5D 俯视、自动寻路 AI、单路推进破墙。

玩家期待的"梁山大军压境"变成了"小股特战队渗透"。游戏体验更接近《盟军敢死队》而非《水浒传》。这不是规模问题，是**情绪调性**问题。

## 1. 改进框架：6 个乐趣杠杆

每个杠杆按 **影响力（高/中/低）× 实现成本（高/中/低）** 标注。

---

## 1.1 杠杆一：填补"等待期"——让玩家始终有事做

> **影响力：高 | 成本：中**

当前最大的问题是"部署完就看戏"。以下是让等待期变得有趣的方案：

### A. 加入"战旗指令"（Command Mode）

**在武将自动战斗的同时，玩家可以下达轻量战术指令：**

| 指令 | 按键 | 效果 | 冷却 |
|------|------|------|------|
| 集火 | Tab + 点击目标 | 所有在场单位优先攻击该目标（3s） | 8s |
| 撤退号角 | R | 所有单位朝部署点方向撤退 3s | 15s |
| 冲锋号角 | F | 所有近战单位移速 +30%（3s） | 12s |
| 散开 | V | 所有单位分散站位（规避 AOE） | 10s |

这些指令不取代自动 AI，而是**短暂覆盖 AI 的优先级**。冷却短（8-15s），让玩家始终有"下一件事"可做。

**为什么好玩**：填补了技能冷却间的操作真空，把"被动观看"变成"主动微操"。4 个指令 × 短冷却 = 额外 20-30 次操作/局，决策密度翻倍。

### B. 战场随机事件

战斗中每 30-45s 触发一个随机事件，迫使玩家即时反应：

| 事件 | 效果 | 玩家应对 |
|------|------|---------|
| "援军抵达！" | 敌方 3 名庄丁从侧门涌入 | 紧急部署肉盾/放 AOE |
| "粮仓着火！" | 一座资源建筑开始燃烧（90s 后自动摧毁） | 分兵先抢收 or 放弃？ |
| "暴雨突降！" | 所有火系效果失效、移速 -20% | 改变战术节奏 |
| "梁山号角！" | 玩家粮草临时 +20（30s 内有效） | 抓住窗口疯狂放技能 |

**为什么好玩**：打破可预测的节奏，创造"灾难→应变→化解"的叙事弧线。即使同一关，每局的事件组合不同，体验永不重复。

## 1.2 杠杆二：重构战斗弧线——从"单峰"到"三幕"

> **影响力：高 | 成本：高（但值得）**

当前战斗是单峰结构。改为三幕式：

### 三幕战斗结构

```
幕一：破门（0-40s）          幕二：巷战（40-90s）         幕三：斩首（90-150s）
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│ 外墙 + 箭塔      │ →  │ 内墙 + 庄丁群 +  │ →  │ 核心 + 守将Boss  │
│ 部署点 1-2       │    │ 资源建筑散布     │    │ + 限时压力       │
│ 战术：选突破方向 │    │ 战术：清场/劫掠   │    │ 战术：all-in      │
│                  │    │ 部署点 3 开启    │    │ 部署点全部开放   │
└─────────────────┘    └─────────────────┘    └─────────────────┘
   低压力                 中压力                  高压力
```

**每幕转变触发：**
- 外墙被破 → 内墙防线激活，瞭望塔开始射击，新部署点解锁
- 内墙被破 → Boss 守将（史文恭/栾廷玉）从核心杀出，核心护盾激活（限时破盾）

**为什么好玩**：
1. 三幕结构自然形成"起承转合"，情绪有高低起伏而非一路平推
2. 每幕解锁新部署点/新机制，维持新鲜感
3. Boss 守将在最后一幕登场，创造"终极对决"的高潮时刻
4. 限时破盾压力让终局紧张刺激

### Boss 守将设计（替代当前 §4.5 的纯建筑核心）

| 关卡 | Boss 守将 | 战斗特点 |
|------|----------|---------|
| L1 祝家庄 | 祝氏三兄弟（依次登场） | 教学型，逐个击破 |
| L2 曾头市 | 史文恭 | 高机动、专杀远程单位，需鲁智深嘲讽吸火 |
| L3 大名府 | 梁中书+护卫队 | 大量召唤杂兵，需公孙胜 AOE 清群 |
| L4 高唐州 | 高廉（术士） | 会驱散+召唤黑风，需时迁潜行打断 |
| L5 蔡京府 | 蔡京（多层护盾）+ 禁军统领 | 终极双 Boss，需完整 combo 链 |

## 1.3 杠杆三：提升"爽感"——水浒该有的视觉冲击

> **影响力：高 | 成本：中（多加特效/动效/Camera 演出）**

2.5D 不等于平淡。以下是低成本高回报的"爽感"注入点：

### A. "英雄时刻"特写

触发条件：英雄在 3s 内完成以下任意一项 → 触发 1.5s 慢镜特写：

| 触发条件 | 演出 |
|---------|------|
| 李逵板斧旋风击杀 3+ 单位 | 屏幕边缘泛红、李逵怒吼特写、伤害数字放大 |
| 花荣百步穿杨秒杀塔/Boss | 镜头跟随箭矢飞行、命中瞬间震屏 |
| 林冲突进破核心最后 10% HP | 黑白闪、林冲特写、"林冲在此！" |
| 鲁智深罗汉金身救援残血队友 | 金身光柱、护盾波纹扩散 |

**为什么好玩**：这不只是视觉糖——它**确认玩家的操作产生了影响**。玩家按下技能→看到史诗级反馈→感到自己很厉害→想再按一次。

### B. 连锁破坏演出

当前设计中的建筑/墙壁是"血条归零→消失"。改为：

- **墙壁**：被李逵破墙 → 木屑横飞、碎片砸伤附近庄丁（环境伤害涌现）
- **箭塔**：被花荣狙击 → 塔身从中间裂开、缓慢倒塌、压死附近单位
- **粮仓**：被劫掠 → 火焰蔓延到相邻木质建筑（连锁反应）
- **核心**：被摧毁 → 建筑逐块崩解、最终轰然倒塌、尘烟弥漫

### C. Combo 播报

当两个英雄技能在 3s 内产生联动效果（如一方的增益被另一方利用），屏幕上方弹出 combo 名称：

| 联动 | Combo 名 |
|------|---------|
| 鲁智深嘲讽 + 公孙胜 AOE 清 3+ | "天雷地火" |
| 吴用召唤 + 李逵嗜血叠满 | "黑旋风·百鬼夜行" |
| 时迁潜行 + 花荣狙杀守核心塔 | "暗度陈仓" |
| 林冲破甲 + 燕青连射 | "枪箭双绝" |

**注意**：Combo 名应该在玩家触发后**首次**弹出（惊喜感），之后只在结算界面统计（不打断节奏）。不要让 combo 名变成"设计师的提示"——让玩家自己发现并命名。

## 1.4 杠杆四：深化"表达感"——我的梁山不一样

> **影响力：中 | 成本：中**

### A. 局内技能变异（每局不同的英雄）

当前：8 将每次完全相同 → 背板后无新鲜感。

改为：每关开始前，每位英雄从 3 个"战场专长"中随机获得 1 个（玩家可见，可选择带不带该英雄）：

| 英雄 | 专长 A | 专长 B | 专长 C |
|------|--------|--------|--------|
| 鲁智深 | 罗汉金身范围 +1 格 | 嘲讽持续 +1s | 受击回血 5%/s |
| 林冲 | 突进距离 +30% | 破甲效果 +20% | 击杀重置冷却 |
| 吴用 | 召唤 3 援军（非 2） | 光环改为全图 | 援军带嘲讽 |
| 公孙胜 | AOE 附带减速 | 雷法对单伤害翻倍 | 雷击后地面残留电网 |
| 李逵 | 破墙系数 ×3（×2→×3） | 嗜血上限 5 层 | 低于 30% HP 时狂暴 |

**为什么好玩**：
- 8 将 × 3 专长 = 24 种变体。每局选 8 携带 → 大量编队 × 专长组合不可能穷举
- "这局吴用抽到了全图光环，那我多带近战"——编队决策和专长产生涌现式联动
- 低成本的 roguelike 轻量化，每局不同但不破坏英雄核心身份

### B. 胜利风格评级（不止三星）

结算时根据本局数据给一个"风格称号"：

| 称号 | 条件 | 含义 |
|------|------|------|
| 智多星 | 用时 > 120s 但兵力损失 < 10% | 稳扎稳打型 |
| 黑旋风 | 李逵击杀数占比 > 40% | 莽夫型 |
| 神箭手 | 花荣/燕青伤害占比 > 50% | 远程型 |
| 鼓上蚤 | 时迁路径最短（没有绕路） | 诡道型 |
| 及时雨 | 鲁智深承伤 > 60% 总伤害 | 坦克型 |
| 天罡星 | 3 星 + 满额拨济 | 完美型 |

**为什么好玩**：让玩家感到"我的打法被系统看见了、被命名了"。三星是客观评分，风格称号是身份认同。两者不冲突，且风格称号天然鼓励玩家尝试不同打法。

## 1.5 杠杆五：重做风险回报——让每次决策有心跳

> **影响力：高 | 成本：中**

当前设计的风险回报存在于"撤兵保星 vs 全押"和"远程点塔 vs 近战破墙"，但都是二选一的安全选择题。需要更刺激的赌博：

### A. "掠阵"机制——用声望赌战力

开战前可选："掠阵"——投入 N 点江湖声望，若 3 星通关则获得 N×2 声望返还；若未 3 星则声望永久损失。

**为什么好玩**：让声望从"被动积累的数字"变成"可以押上的筹码"。高风险高回报的赌博感。

### B. "劫营"——高风险高回报部署

除了安全区部署点外，某些地图提供 **1 个"劫营点"**：位于敌方腹地（如粮仓旁），部署在此处的单位立即获得攻击力 +30%、移速 +20%，但初始吸引所有塔的仇恨。

**为什么好玩**：高风险部署创造戏剧性。玩家可能在开局直接劫营李逵→李逵秒破内墙→敌军全压→鲁智深紧急嘲讽救援→观感拉满。也可能劫营失败→直接崩盘→复盘学教训。

### C. "遗计"——死后还有用

当一个英雄阵亡时，其"遗计"自动触发（每位英雄不同）：

| 英雄 | 遗计 |
|------|------|
| 鲁智深 | 阵亡地形成伤害区域（3s） |
| 林冲 | 给周围敌人上破甲 |
| 吴用 | 召唤 2 名亡语庄客（不计入上限） |
| 公孙胜 | 周围降下小型雷暴 |
| 花荣 | 射出最后一箭（高伤害） |
| 时迁 | 掉落一个钱袋（劫掠收益 +5%） |

**为什么好玩**：把"失败"（英雄阵亡）变成"有代价的机会"。阵亡不再只是惩罚，而是需要计算的战略要素——"林冲还剩 10% HP，要不要让他冲进去吃伤害触发群体破甲？"

## 1.6 杠杆六：扩展社交与重玩——战斗不止一场

> **影响力：中 | 成本：高（适合长期路线图）**

### A. 异步"守寨"模式

玩家可以用自己解锁的防御建筑（塔/陷阱/墙）设计自己的山寨布局，其他玩家可以来"劫寨"。防守成功获得声望。

这不需要实时 PvP，而是：
1. 玩家 A 设计布局 → 上传
2. 玩家 B 挑战 → AI 控制防御
3. 双方获得排名分和奖励

**为什么好玩**：这是 Clash of Clans 核心乐趣的轻量化复刻——"我设计的防御能不能挡住别人？"创造无限的内容生成循环（UGC）。

### B. 周常挑战

每周刷新一个"限定挑战"：

| 挑战类型 | 示例 |
|---------|------|
| 英雄池限制 | "本周只能使用 5 名武将" |
| 机制修改 | "所有单位移速翻倍" |
| 禁用机制 | "禁用所有远程武将" |
| BOSS RUSH | "连续挑战 3 关，血量继承" |
| 逆向挑战 | "用守军阵容防守，AI 自动进攻" |

排行榜 + 赛季奖励。**为什么好玩**：给已经通关的玩家一个"回来再玩一次"的理由，同时通过限制条件逼迫玩家发现新的 combo（涌现引擎 x 限制条件 = 创意爆发）。

## 2. 改进方案优先级矩阵

```
影响力
 高 │  杠杆一（战旗指令）        │  杠杆二（三幕战斗弧线）
    │  杠杆一（战场随机事件）    │
    │  杠杆五（掠阵/劫营/遗计）  │
    │  杠杆三（英雄时刻/连锁演出）│
    │                           │
 中 │  杠杆四（技能变异/风格称号）│  杠杆四（技能变异）
    │  杠杆三（Combo 播报）      │  杠杆六（守寨模式/周常）
    │                           │
 低 │                           │
    │                           │
    └───────────────────────────┴─────────────────────
              中成本                    高成本
```

## 3. 如果只做三件事（最小可行改进）

按"性价比"排序，建议优先落地：

### 🥇 第一优先：战时指令（杠杆一.A）

**投入**：在现有 `RealTimeBattleManager` 上加 4 个全局指令（集火/撤退/冲锋/散开），每个指令本质是"短暂覆盖所有单位的索敌/移动 AI"。代码改动量：约 200-300 行逻辑层 + UI 4 个按钮。

**收益**：决策密度从 ~28 次/局提升到 ~50+ 次/局，玩家从"等待冷却"变成"持续微操"。这是 ROI 最高的单点改进。

### 🥈 第二优先：三幕战斗结构（杠杆二）

**投入**：重构关卡数据（墙 HP / 触发条件 / 阶段事件），新增 Boss 守将 AI。对现有防御系统有侵入性改动。

**收益**：根治"乐趣曲线衰减"问题。三幕结构天然制造叙事弧线，Boss 守将解决"摧毁核心太抽象"的体验空洞。这是**对战局体验质量提升最大**的改动。

### 🥉 第三优先：英雄时刻 + 连锁破坏演出（杠杆三）

**投入**：Camera 特写系统 + 粒子特效 + 建筑物理破碎动画（可复用 Three.js 已有资源）。

**收益**：解决"水浒幻想落空"问题。让 2.5D 也有视觉冲击力，让玩家感到"我在指挥梁山好汉"而非"我在移动棋子"。

---

## 4. 一句话总结

当前设计**策略骨架好**（资源分离、墙博弈、状态组件），但**血肉不足**（操作密度低、节奏平、缺乏高潮、缺少水浒的"燃"）。

三步改造：
1. **让手忙起来**——战时指令填补等待真空
2. **让心跳起来**——三幕结构 + Boss 守将制造情绪弧线  
3. **让血热起来**——英雄时刻 + 连锁破坏兑现水浒幻想

三样加在一起，这个战斗系统就能从"合格的策略解谜"变成"让人想再来一局的动作策略体验"。
