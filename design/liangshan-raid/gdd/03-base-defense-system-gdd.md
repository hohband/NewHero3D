# GDD-03 基地与防御系统（Base & Defense System）

> **所属模式**：劫寨战斗模式（Raid the Stronghold）· Phase 2 系统设计
> **上游文档**：`../combat-mode-design.md`（DS-001，§4 基地结构、§6 关卡递进、§0.1 已审批锁定）
> **设计支柱对齐**：P3 地形博弈（读图 > 莽）、P1 以少胜多（纵深防御）、R1/R2（寻路性能）、R4（差异化谜题化）
> **架构红线**：防守方为静态布局 + AI 守军响应（真·CoC 异步对抗），核心/胜负判定在逻辑层；**不采用 SRPG submit_command 回合模型**。寻路用流场（Flow Field，R2），随机偏移经 `RollSource` 注入。

---

## ① 概述（Overview）

基地与防御系统构建敌方山寨（祝家庄等）的**墙—塔—陷阱—庄丁—Boss 守将—核心**体系，是 P3「地形博弈」的载体。v2 关键升级：防守方从"静态布局 + 触发式 AI"升级为**动态防御（警报等级/哨兵/巡逻队/指挥官 AI）+ 战争迷雾 + 环境系统**，让"对面有脑子"（combat-mode §0.2、§4.6–§4.8）。核心由 **Boss 守将守卫**，击败 Boss 后核心方可摧毁（§4.9）。

- 模板 + **结构随机**（v2）：每局不仅偏移坐标，更随机选"墙木/石""塔位箭塔/瞭望塔"，避免"随机偏移纯 cosmetic"。
- 资源建筑外围散布，诱导"为劫掠而分兵"的风险回报（P4 战场体现，v2 劫掠为实时动词，详见 GDD-04）。
- **v2 同步**：动态防御 / 信息战 / 环境 / Boss 守将 / 主动欺骗 —— 见 `../combat-mode-design.md` §0.2 / §4.6–§4.9。

## ② 机制（Mechanics）

1. **结构层级**
   - 外墙（木，可破坏阻挡）/ 内墙（石，更硬）/ 箭塔·瞭望塔·法术塔（L4+）/ 陷阱（陷坑·绊马索·火油）/ 庄丁守军（**v2 6 兵种**）/ **Boss 守将（v2）** / 核心（忠义堂）。
2. **墙阻挡与破墙**
   - 墙 = 可破坏障碍，独立 HP；默认单位**绕墙寻路**找门/缺口。
   - 目标被墙阻挡且最近门过远 → 转入**拆墙模式**攻击墙；破墙专精单位/技能（李逵板斧旋风 ×2、林冲破甲）优先破墙。
   - **墙不阻挡远程**：塔、花荣、燕青可从墙外射击墙内 → "远程点塔（安全慢）vs 近战破墙（快但脆）"核心博弈。
   - **v2 流场分块重算（H1）**：墙被完全拆毁时只重算墙所在分块及其下游分块（Dijkstra 流场 + 增量更新），避免全图重算卡顿。
3. **塔索敌**
   - 箭塔（range 5，**DPS 15**）/ 瞭望塔（range 7，**DPS 8**）/ 法术塔（range 6，L4+，**DPS 5**，**v2 驱散改为周期脉冲**）。
   - 受鲁智深嘲讽时优先攻击嘲讽目标（与 GDD-02 联动）。
   - 塔自身 HP 有限，可被花荣/燕青远程点掉。
4. **陷阱**：触发式区域，踩中触发伤害/控制；时迁「妙手」经过不触发（直接拆除）；**v2 妙手扩展为暗杀哨兵**（§②.6）。
5. **庄丁（v2 6 兵种分化）**：枪兵（反突进）/ 弓手（点杀低血施法者）/ 哨探（逃跑报信）/ 力士（守核心不追）/ 火攻手（投火油造燃烧地面）/ 医师（治 Boss）。与英雄形成石头剪刀布涌现（AI 行为自然产生，非硬编码克制）。
6. **核心判定（v2）**：核心由 **Boss 守将守卫**，击败 Boss 后核心护盾消失方可摧毁 = 进攻方胜；若进攻方兵力耗尽且不可再部署 = 进攻方败。
7. **模板随机偏移**：多套布局模板，每局注入 `RollSource` 随机偏移/旋转 + **结构随机**（墙质/塔种），保证可解、每局不同；最多重 roll 10 次，仍不可达则 ±1 格噪声（H4）。
8. **v2 动态防御（警报等级，§4.6）**：敌方基地 0–4 级警报（只升不降）。0 平静→1 可疑（哨兵警觉）→2 警戒（庄丁出动/援军）→3 激战（Boss 出击/法术塔激活/陷阱全开）→4 绝望（焦土）。玩家无法消警报，只能选低等级时做多少事。
9. **v2 哨兵 / 巡逻队（§4.7）**：哨兵固定观察（扇形 120°/5 格，可被时迁静默暗杀、花荣狙杀）；巡逻队 2–3 人沿随机偏移路线循环，发现敌→不攻击先跑回报警（信息节点），受鲁智深嘲讽可拉住防报警。
10. **v2 敌方指挥官 AI（§4.6）**：每 15–20s 决策一次，消耗指挥点(CP)，基于**它知道的信息**（受战争迷雾限制，非上帝视角），偏好因关卡主将性格而异（史文恭集火远程/高廉驱散…）。
11. **v2 主动欺骗（§4.6）**：玩家可放"诱饵"（草人）引塔/守军偏离防线，或"佯退"引守军追击再回马枪；前提守军 AI 真的会追诱饵/离防线。
12. **v2 战争迷雾（§4.7）**：战场初始被雾覆盖，仅己方视野（4–6 格）可见；侦查期 8s 全图短暂可见，结束后恢复；侦查期标记永久保存。
13. **v2 环境系统（§4.8）**：天气（晴/雨/雾/风/雪，开局随机）+ 昼夜（黑夜 toggle）+ 可互动物（火药桶/酒坛/粮草堆/滚石/烽火台/吊桥），每物体 ≥2 种互斥用法。
14. **v2 Boss 守将（§4.9）**：具名反派守核心，同时是指挥官 AI 载体；击败→核心可毁；可招降为将（耗声望+拨济+10%）。

## ④ 公式（Formulas）

- **破墙**：`Wall_HP_remaining = Wall_HP − Σ(breach_dps_i × breach_coeff_i × dt)`
  - 李逵板斧旋风：`breach_coeff × 2`（GDD-02）。
- **塔伤害**：选最近敌方单位（或嘲讽目标），`Damage = tower_dps × dt`。
- **减速（瞭望塔）**：命中目标 `move_speed × (1 − slow_pct)`，持续。
- **陷阱触发**：单位进入陷阱格 → 触发伤害/控制；时迁妙手 → 拆除不触发。
- **核心判定（胜，v2）**：`Boss_HP ≤ 0`（守卫倒下，核心护盾消失）且 `Core_HP ≤ 0 ⇒ 进攻方胜`。
- **进攻方败**：`Core_HP > 0` 且 `兵符剩余 < min_cost` 且 `粮草耗尽` 且 `场上存活 = 0`（即无可再部署能力）。
- **v2 警报等级**：`alert_level` 只升不降；触发条件见 `alert_config.csv`；影响哨兵/巡逻/塔/援军/Boss 行为。
- **v2 驱散脉冲（dispel_config）**：每 `pulse_interval=8s` 持续 `pulse_duration=1s`、范围 `range=3` 内清空增益光环（**不清召唤物**）；玩家可在脉冲间隙放技能或放诱饵骗出。
- **v2 指挥官决策**：每 15–20s 消耗 CP 从动作池选效用最高行动；信息受战争迷雾限制。
- **模板偏移（v2 结构随机）**：`actual_pos = template_pos + RollSource.uniform(−offset,+offset)` 或旋转 θ + 结构随机；偏移须满足**核心可达性校验**，最多重 roll 10 次，否则 ±1 格噪声（H4）。

## ⑤ 边缘情况（Edge Cases）

1. **墙被完全拆毁**：寻路重算（流场更新，R2），单位改走缺口。
2. **核心被远程越墙攻击**：允许（墙不阻挡远程），核心独立 HP 可被花荣狙。
3. **陷阱与墙重叠**：拆墙同时踩陷阱 → 二者独立结算，先判定陷阱触发再拆墙。
4. **庄丁全部阵亡**：守军不再触发，核心裸防。
5. **模板偏移致核心不可达**：`RollSource` 偏移须约束在可达范围，校验核心路径存在，否则重 roll（最多 10 次，H4）。
6. **v2 法术塔驱散脉冲**：周期脉冲（8s/1s，范围 3）清空增益光环、**不清召唤物**；时迁潜行态被脉冲命中→显形（与 GDD-02 联动）。可放诱饵骗出脉冲窗口。
7. **同格多单位**：索敌/伤害按列表遍历，无重复/遗漏结算。
8. **v2 战争迷雾**：迷雾中敌人真实存在并行动，仅不可见；哨兵/巡逻队脚步声方向性音频提示；侦查期标记永久可见。
9. **v2 警报只升不降**：玩家无法消警报；Boss 在警报 3 才出击，低等级潜入可抢先布阵。
10. **v2 主动欺骗**：诱饵吸引塔/守军偏离防线；佯退引守军追击，须守军 AI 真实响应。

## ⑥ UI 接口（UI Interface — 对齐 §7.5 / §7.6 HUD）

- **侦查期（层 3 可折叠，v2 永久保存）**：标记墙/塔/陷阱位置与类型；旋转缩放读图（§7.6）；**标记永久保留至战斗结束**。
- **战斗 HUD 层 1**：敌方核心血量条 + **Boss 守将血条** + **警报等级**常驻。
- **v2 战争迷雾表现**：迷雾 shader 覆盖；哨兵提灯/烽火为可见光点；方向性脚步声。
- **即时反馈**：破墙粒子、塔火弹道、陷阱触发特效（满足发现感/挑战感）；**v2 高光时刻 Juiciness**（8 英雄招牌瞬间，combat-mode §0.2.2）。
- **状态图标（层 2）**：减速/嘲讽/驱散/潜行/警报/关系羁绊显示在受影响单位头顶。

## ⑦ 依赖（Dependencies）

- **跨系统**：
  - GDD-01 部署系统：部署点/劫营点位置、单位寻路受墙阻挡、在场上限、撤兵/直控。
  - GDD-02 武将技能系统：嘲讽改塔索敌、破墙系数、L4 驱散脉冲克制增益（不清召唤）。
  - GDD-04 经济与评分系统：资源建筑=劫掠目标、建筑总数/摧毁数（口径含墙+塔+资源+核心，M7）、核心/Boss HP 供评分。
- **Phase 3 接口**：
  - `RealTimeBattleManager` + 流场寻路（R2，**v2 分块重算**，H1）；`RollSource` 注入模板偏移/结构随机/闪避/陷阱（§8）。
  - **v2 新增逻辑模块**：动态防御 AI（警报/哨兵/巡逻/指挥官）、战争迷雾（visibility_system）、环境状态机（weather/environment_objects）、Boss 守将 AI、主动欺骗（诱饵/佯退索敌）。
  - 核心/Boss/胜负判定在逻辑层统一广播 `battle_end_event`（供 GDD-04 评分）。
  - **非 SRPG submit_command**：防御为动态布局 + 触发式/响应式 AI（v2 升级）。

## ⑧ 验收标准（Acceptance Criteria）

- [ ] 墙可破坏、不阻挡远程、破墙专精 ×2 生效。
- [ ] 塔索敌优先级 + 嘲讽改向正确。
- [ ] 陷阱触发 / 时迁妙手拆除正确。
- [ ] 核心被毁=胜、兵力耗尽=败判定准确。
- [ ] 模板随机偏移 + **结构随机**保证可解且每局不同（无背板，H4 重 roll 上限）。
- [ ] **v2 流场分块重算**：墙破仅重算相关分块，无全图卡顿（H1）。
- [ ] **v2 动态防御**：警报 0–4 只升不降、哨兵/巡逻/指挥官 AI 按玩家行为反应（"对面有脑子"）。
- [ ] **v2 战争迷雾**：局部可见、侦查期标记永久保存、方向性音频。
- [ ] **v2 环境系统**：天气/可互动物交互正确，连锁反应自动产生。
- [ ] **v2 L4 法术塔驱散=周期脉冲（8s/1s，范围3，不清召唤，可骗出）**，非硬 counter。
- [ ] **v2 Boss 守将：击败后核心可毁；招降逻辑正确（耗声望+拨济+10%）**。
- [ ] 资源建筑外围散布可被劫掠并计入评分（口径 M7）。
- [ ] CSV 字段完整，数值热更后实时生效。
